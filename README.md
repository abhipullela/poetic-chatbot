# Poetic Chatbot

Poetic Chatbot is a Python-based AI chatbot developed in Google Colab that generates creative and poetic responses through interactive conversations. 
The project focuses on natural language processing concepts, text generation, and user interaction using a simple and modular approach.

## Features
- Interactive chatbot conversation
- Poetic and creative text generation
- NLP-based response handling
- Google Colab implementation
- Simple and modular code structure

## Technologies Used
- Python
- Google Colab
- Natural Language Processing (NLP)

## Files
- `Poetic_Chatbot.ipynb` – Main notebook containing the chatbot implementation
- `Poetic_Chatbot_Report.docx` – Project documentation/report

## Objective
The objective of this project is to demonstrate chatbot development and creative text generation using Python and NLP concepts in an interactive notebook environment.
